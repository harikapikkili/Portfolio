# Portfolio Overview

This portfolio showcases the skills, projects, and academic achievements of Vijaya Lakshmi Jyothi, a passionate iDigital Entrepreneur and Web Developer. The portfolio is designed to provide a comprehensive view of my professional journey, including my educational background, certifications, and hands-on projects.

## Project Structure

The project consists of the following files:

- **index.html**: The main landing page that links to all other sections of the portfolio.
- **academic.html**: Contains detailed academic information, including degrees, institutions, and scores.
- **certificate.html**: Lists various certifications obtained, with links to view each certificate.
- **projects.html**: Showcases completed projects with descriptions and links to view them.
- **skills.html**: Details technical skills and experience, including internships and relevant courses.
- **contact.html**: Provides contact information and a form for visitors to send messages.
- **style.css**: Main CSS file that defines the layout, colors, and typography for the portfolio.
- **style2.css**: Additional CSS styles for specific sections or components of the portfolio.
- **PDF Certificates**: Various PDF documents containing certificates from Accenture, Infosys, and NPTEL.

## Setup Instructions

1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to view the portfolio.
3. Navigate through the links to explore different sections of the portfolio.
