# 🌐 Personal Portfolio Website

Welcome to my personal portfolio! This website highlights my education, technical skills, certifications, projects, and professional journey. It serves as a central place to showcase my work, achievements, and experience as a Web Developer and aspiring Software Engineer.

## 🚀 Features

* 🏠 Responsive Home Page
* 👩‍🎓 Academic Qualifications
* 💻 Technical Skills
* 📂 Project Showcase
* 📜 Certifications
* 📞 Contact Form
* 📱 Mobile-Friendly Design
* 🎨 Clean and Modern User Interface

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

2. Open the project folder

```bash
cd portfolio
```

3. Launch the website

Simply open **index.html** in your preferred web browser.
